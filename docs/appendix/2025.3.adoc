== 2025.3

=== 2025.3.0

Yes, still in the 2025.x series, and still no changes that warrant a major version bump increase. Starting with this release we will build on JDK 26 by default, still releasing for Java 17. All our testing will be exercised on 17, 21, 25 and 26.

This release brings support for the Cypher 25 `SEARCH` clause as well as for `VECTOR` literals. Together you can now query Neo4j vector indices without using procedures. Obviously, this needs a modern Neo4j version to run.

==== 🚀 Features

* Add support for vector literals and the `SEARCH` clause. (#1548)

==== 🔄️ Refactorings

* Fire different events for expressions created from within `RETURN` or `WITH` clauses. (#1544)
